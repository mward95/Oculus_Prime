Axial length vs Patient ID

With the first figure what we are trying to identify was who the target group for treatment would be. When looking at the data set we wanted to find which child had myopia vs which child did not have myopia.

Each bubble repsents the axial length of each child, as the bubble gets bigger that means the child has a longer eye. As the bubble gets smaller it means the child has a shorter eye

From the data set we found 72 childen where classified as having myopia out of 619 children in total.

Which parent passed the trait?

From this graph we are able to be more granular with the data and focus on those that truly have poor eye sight. With myopia being a genetic trait we wanted to to see if the trait was focused more on a specfic parent or both parents.

Both parents are myopic contains 33 children

Each bubble repsents the axial length of each child, as the bubble gets bigger that means the child has a longer eye. As the bubble gets smaller it means the child has a shorter eye

just mom is myopic has 18

just dad is myopic has 21

Some interesting results we found where that its practically 50/50 if one parent has the trait and the other parent does not have the trait. While if both parents have the trait the likely hood of being myopic goes way up.

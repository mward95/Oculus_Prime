Summary

What We Learned

What was learned from this project was how to produce an analytical model in Python that after being cleaned, normalized, pretrained and then had its hyperparameters tuned could be used to create future analysis. The relational databases we used where RDS from AWS which were then converted to a JSON. The models created showed meaningful results with predictive power over 80%.

Where We Are Going

Where we would like to take this project further with more time. We would like to get closer to the upper 90's in our model accuracy so that it could be looked at as a model for real research. To collect more and newer data so that we could further train the model more.
